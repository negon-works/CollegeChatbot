Bot Management Guide

Before adding a new bot entry, remember:

  • Select Category: Choose the correct category before saving.
  • Chatbot - greetings, bot info, system responses
  • College - courses, admissions, hostel, campus details
  • Add Clear Triggers: Triggers are the words users type to activate a response.
  • Set Priority Properly: Lower number = higher priority (matches first).
  • Check Active Status: Make sure the entry is Active if you want the bot to use it.

For detailed instructions, please read the Full Bot Management Guide.

1. Adding or Editing Content

Open Bot Content to add, edit, or manage chatbot responses.

Each entry contains:

  • Triggers: The words or phrases users type that activate the response.
  • Response: The message the chatbot will send to the user.
  • Priority: Controls which response should match first.

2. Category Selection (Important)

When creating a new entry, always select the correct Category.

This helps keep the bot organized and makes it easier for different teams to manage content.

Available Categories

  • Chatbot: Used for general bot behavior such as greetings, identity, thanks, fallback messages, etc. Managed mainly by chatbot designers or developers.
  • College: Used for college-related information such as courses, admissions, hostel details, fees, documents, and campus information. Managed mainly by college staff or administrators.

Selecting the correct category ensures that content is sorted and easy to manage for both teams.

Important: Do not forget to select a category before saving a new entry.

3. Understanding Priority

Priority determines which trigger is matched first. Lower numbers have higher priority.

Correct Configuration

hello priority 10

admission priority 5

User: hello admission plz

Bot reply: Admission information

Incorrect Configuration

hello priority 5

admission priority 10

User: hello admission plz

Bot reply: Greeting message

4. Active / Inactive Toggle

  • Active: The chatbot can use this entry.
  • Inactive: The entry is temporarily disabled.

This allows staff to quickly enable or disable responses without deleting them.

5. Best Practices

  • Keep triggers short and clear.
  • Avoid duplicate triggers across entries.
  • Always check priority order before saving.
  • Select the correct Category (Chatbot or College) to keep the system organized.

Properly organizing entries will help the chatbot provide accurate and reliable responses to users.
